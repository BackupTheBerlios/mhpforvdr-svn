MHP API list

I recommend to read the main parts of the MHP specification first to get a general idea (chapters 0, 5, 9, 11, 13). Some APIs are mentioned at various places, giving clarifications or extensions. Once you have understood the basic architecture, you will know what most of the APIs in this list are supposed to do.

The second column gives the first page where the respective package is described in the main part of the MHP specification (mostly in chapter 11), while the third column gives the location of the actual specification of that package (mostly in the annexes)

If not noted otherwise (Sun, Davic), all references point to the MHP specification. First the section numberis given, then the page number. These page numbers are taken from the MHP specification (ETSI TS 102 812) version 1.1.1 from 2001-11. Other versions of this specification will have different numbers, in this case use the section number.

Some sources for the specifications are listed below the table.

The packages listed here vary extremely in extent (from one class to more than a hundred) and difficulty to implement (some contain only trivial interfaces, other require heavy-weight native support). So do not compare the number of packages with green status to those with red status, the percentage you get is meaningless.
